## Welcome to Articley        
Articley is a website where you can post articles, write articles, delete articles.          
Articley is kind of like Wikipedia when I think about it.       
Articley also lets you write articles in Markdown.       

# Tech stack 
The tech stack for this is Appwrite, Appwrite pages, Appwrite Tables, and HTML and that's all!      
Although this stack is simple, I think it's quite powerful.
